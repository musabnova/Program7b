# Program 7b - JSP Cookie Management (Form Based)

## Objective
Create a JSP program to:
- Accept Name, Domain and Max Expiry Age
- Add cookie using form
- Display active cookie list

## Files
- index.jsp
- addCookie.jsp
- showCookies.jsp

## Source Code
- index.jsp: https://github.com/musabnova/Program7b/blob/master/src/main/webapp/index.jsp
- addCookie.jsp: https://github.com/musabnova/Program7b/blob/master/src/main/webapp/addCookie.jsp
- showCookies.jsp: https://github.com/musabnova/Program7b/blob/master/src/main/webapp/showCookies.jsp

## How to Run
1. Run project on Apache Tomcat
2. Open: http://localhost:8080/Program7b/
3. Enter Name, Domain, Expiry time
4. Click "Add Cookie"
5. Click link to view active cookies
